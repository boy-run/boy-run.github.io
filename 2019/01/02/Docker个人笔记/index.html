Docker个人笔记

Posted on 2019-01-02

镜像篇

获取镜像

到docker hub下载 或者 docker search xxx

docker pull xxx

查询本地镜像：

docker images

删除本地镜像：

docker rmi xxx

容器篇

查看运行容器

docker ps

查看所有容器

docker ps -as

运行容器

docker start -i xxx

进入容器

docker exec -it 7f6dca80b660 /bin/bash

停止容器

docker stop xxx

停止所有容器

docker stop $(docker ps -q)

删除容器

docker rm -f xxx

删除所有容器

docker rm $(docker ps -aq)
