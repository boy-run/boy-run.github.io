Docker可以让开发者打包他们的应用以及依赖包到一个轻量级、可移植的容器中，然后发布到任何流行的Linux机器上，也可以实现虚拟化。容器是完全使用沙箱机制，相互之间不会有任何接口（类似iPhone的app）,更重要的是容器性能开销极低。

镜像篇

  1. 增加镜像：到docker hub下载 || docker search

  2. 查询本地镜像：docker images

  3. 删除本地镜像：docker rmi xxx

容器篇

  1. 查看运行容器：docker ps
  2. 查看所有容器：docker ps -as
  3. 运行容器：docker start -i xxx
  4. 进入容器：docker exec -it 7f6dca80b660 /bin/bash
  5. 停止容器：docker stop xxx
  6. 停止所有容器：docker stop $(docker ps -q)
  7. 删除容器：docker rm -f xxx
  8. 删除所有容器：docker rm $(docker ps -aq)
